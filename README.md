# Tower-of-Guild

1. Notepad++ or an equivilant organized txt document program
2. The base server files: https://mega.nz/#!IR9ngIbQ!mkDqoYmV5eKSYKemTwnZx1wwwxbJhIBC7Nalc-6AF24
